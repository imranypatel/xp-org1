## How to Use This Tracker

1. **Team Lead** adds a row for each team member when they begin the LaaS program
2. **Team member** updates their own row when they pass a module assessment
3. **Team Lead** counter-signs L3 completion and updates the Aggregate Summary
4. For L3, link each portfolio evidence item (GitHub URL or docs/ path) in the Evidence column

Passing thresholds:
- L1: Module 1 quiz ≥ 70%
- L2: Module 2 Section A ≥ 70% + both practical exercises submitted
- L3: All 4 portfolio evidence items at Proficient (3) or above in all rubric criteria
