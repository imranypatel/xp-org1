## How to Use This Tracker

1. **Prerequisites**: All team members must complete ST Foundations L1 before starting this program.
   Record ST L1 status in the `ST L1` column.

2. **Level progression**: A member advances only when *both* the quiz threshold *and* the practical
   gate are met. Update the status after confirming both.

3. **Status codes**:
   - `—` Not started
   - `🔄` In progress
   - `✅` Passed (quiz + practical gate)
   - `❌` Attempted, not yet passed — see notes

4. **Rubric ratings**: Use D (Developing), P (Proficient), M (Mastery) from the Competency Rubric.
