## Why This Training Exists

Every team member learns continuously — new tools, new domain knowledge, new processes. Yet most learning is approached as a linear transaction: study more → know more. This linear model misses the structural dynamics that determine whether learning sticks, compounds, or stalls.

Learning is a system. It has stocks (accumulated knowledge, skills, motivation, confidence), flows (study rate, forgetting decay, practice intensity), reinforcing loops (the competence loop: skill → success → confidence → motivation → effort → skill), balancing loops (the gap-closing loop), and delays (effort before mastery). When you can see these structures, you can design your learning instead of just doing it.

At the organizational level, the team's collective knowledge stock has its own dynamics: concentration risks (bus factor), diffusion mechanisms (documentation, mentoring), and cultural loops that either accelerate or stall the organization's learning capacity.

> **The core insight**: Sustainable learning is not about working harder. It is about designing the system — reducing the outflows (forgetting), strengthening the inflows (deliberate practice), and protecting the reinforcing loop (competence → confidence → motivation) from being disrupted by time pressure and discouragement.

## Program Structure

Three progressive levels. Each builds on the previous. Level-up requires both a quiz pass **and** practical application evidence.

```
L1 The Learning System    →    L2 Learning System Design    →    L3 Organizational Learning
     (3–4 hours)                     (6–8 hours)                  (10–12 hrs + portfolio)
 ST Foundations L1             L1 quiz ≥ 70%               L2 pass + 2 applied analyses
```

---

### Level 1 — The Learning System

**Goal**: Build the mental model that learning is a system with stocks, flows, loops, and delays — not a linear process. Diagnose your current learning dynamics using ST vocabulary.

**Who needs it**: Every team member. L1 is the entry point for all subsequent levels.

**Learning outcomes**:
- Identify the key stocks and flows in your personal learning system
- Explain the Competence Reinforcing Loop and describe how it can become vicious
- Recognize the Gap-Closing Balancing Loop and predict when effort will naturally reduce
- Explain why consistent low-intensity practice accumulates more stock than occasional high-intensity bursts
- Identify a delay in your own learning and describe its effect on behavior

**Duration**: ~3–4 hours self-paced

**Content**: [Module 1 — The Learning System](./module-1-introduction.md)

**Assessment**: [Module 1 Quiz](./assessments/module-1-quiz.md) — 10 scenario questions, passing threshold 70%

---

### Level 2 — Learning System Design

**Goal**: Recognize system archetypes in learning failures. Design structural interventions for personal and team learning. Apply Meadows leverage points to learning challenges.

**Who needs it**: Anyone who wants to improve their own learning efficiency or contribute to team learning practices.

**Prerequisites**: Module 1 quiz ≥ 70%

**Learning outcomes**:
- Identify the Limits to Growth, Fixes that Fail, Shifting the Burden, and Eroding Goals archetypes in personal and team learning scenarios
- Design a personal learning system with explicit inflow strengthening and outflow reduction strategies
- Explain Senge's 5 disciplines through a stock-flow lens
- Distinguish single-loop from double-loop learning (Argyris) and explain when each is appropriate
- Identify Meadows Level 2 (paradigms), Level 4 (rules), and Level 6 (information flows) leverage points in a learning challenge
- Draw a CLD of a personal or team learning system

**Duration**: ~6–8 hours self-paced

**Content**: [Module 2 — Learning System Design](./module-2-practitioner.md)

**Assessment**: [Module 2 Quiz](./assessments/module-2-quiz.md) — 8 scenario questions + 2 practical exercises

---

### Level 3 — Organizational Learning

**Goal**: Model the organizational learning system. Identify structural vulnerabilities. Design high-leverage interventions. Facilitate team learning retrospectives.

**Who needs it**: Team leads, senior contributors, HR domain members, anyone responsible for team capability.

**Prerequisites**: Module 2 pass + ≥ 2 real learning challenge ST analyses in GitHub issues (linked evidence)

**Learning outcomes**:
- Map the organizational knowledge stock: distributed structure, concentration risks (bus factor), transfer flows
- Identify the Learning Culture Reinforcing Loop and the conditions that weaken it
- Explain how competency development in one domain creates positive feedback into adjacent domains
- Design a structural intervention for a persistent organizational learning failure
- Facilitate a 60-minute "Our Learning System" retrospective with a team
- Coach a colleague in designing their personal learning system using ST

**Duration**: ~10–12 hours self-paced + portfolio assembly

**Content**: [Module 3 — Organizational Learning](./module-3-mastery.md)

**Assessment**: [Module 3 Portfolio Assessment](./assessments/module-3-quiz.md) — evidence checklist, no timed quiz

## How This Program Relates to ST Foundations

This program applies the tools you learned in ST Foundations to a specific domain: learning itself. The relationship is:

| ST Foundations teaches... | Learning as a System applies it to... |
|--------------------------|--------------------------------------|
| What a stock is | Knowledge stock, skill stock, confidence stock |
| What a reinforcing loop is | The Competence Loop (virtuous and vicious) |
| What a balancing loop is | The Gap-Closing Loop |
| What delays do to behavior | Why effort ≠ immediate mastery; why cramming fails |
| What Limits to Growth looks like | Learning plateaus under cognitive load |
| What Fixes that Fail looks like | Cramming, AI over-reliance |
| What Meadows leverage points are | Level 2: growth mindset as paradigm; Level 6: visible progress |

### Canonical texts

- Peter Senge, *The Fifth Discipline* (Doubleday, 1990) — the organizational learning system; mental models, team learning, shared vision
- Donella Meadows, *Thinking in Systems* (Chelsea Green, 2008) — structural lens for all analysis
- Chris Argyris, *Double Loop Learning in Organizations* (Harvard Business Review, 1977) — single vs. double-loop
- Hermann Ebbinghaus, *Über das Gedächtnis* (1885) — the forgetting curve; foundation of spaced repetition
- Cal Newport, *Deep Work* (Grand Central, 2016) — deep vs. shallow processing as outflow-rate regulators
